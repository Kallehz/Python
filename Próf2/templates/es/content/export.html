Incluir DTD en la descripción XML
Crear archivo con un único fichero XML

Nota: Este comando crea un archivo que contiene:
  • un fichero XML que describe la estructura y contenido de esta carpeta
  • todos los ficheros incluidos en la carpeta

Ayudas:

  • Puedes editar el contenido del archivo e importar de nuevo a esta carpeta
  • Usa el DTD para pre-validar la descripción XML. Aunque la carpeta esté vacía, el DTD reflejará su estructura global
