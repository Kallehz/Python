Bienvenido a Mooshak, un sistema creado para manejar concursos de programación en red. Mooshak permite la evaluación automática de los programas enviados, la resolución de cuestiones sobre los problemas, y el seguimiento de las impresiones de código.

Use este enlace para entrar en Mooshak. Existen 4 perfiles de usuario que requieren autentificación. Esta distribución incluye un concurso de prueba (Contest) con un usuario definido para cada perfil. Los nombres de los usuarios y las claves coinciden con el nombre del perfil correspondiente. Puedes (y debes) crear nuevos usuarios y cambiar las claves después, especialmente la clave del usuario admin. También puedes usar otras características ofrecidas por Mooshak, tales como
  1. Resultados de los concursos
  2. Registrarte en un concurso on-line
  3. Ayuda on-line
  4. Listas de comprobación
