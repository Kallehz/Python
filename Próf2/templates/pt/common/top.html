Bem vindo ao Mooshak, um sistema para gerir concursos de programação na web. O Mooshak suporta avaliação automática dos programas submetidos, resposta a questões a seguimento de impressões.

Use este apontador para entrar no Mooshak. Há 4 perfis de utilização requerendo autenticação. Esta distribuição inclui um concurso de teste com um utilizador definido em cada perfil. Os nomes de utilizador e senhas são os mesmos do nome do perfil. Pode (e deve) criar novos utilizadores e alterar estas senhas posteriormente., especialmente a senha de admin. Pode também utilizar outras funcionalidades do Mooshak, tais como
  1. Resultados de concursos
  2. Registar em concursos em-linha
  3. Ajuda em linha
  4. Listas de verificação

Pode também usar o Mooshak como um serviço web REST. Use esta página de teste para ver como o serviço é usado.
