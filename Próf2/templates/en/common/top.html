Welcome to Mooshak, a system for managing programming contests on the Web. Mooshak suports automatic judging of submitted programs, answering questions about problems and tracking printouts.

Use this link to login in Mooshak. There are 4 user profiles requiring an authentication. This distribution includes a test contest (Contest) with an user defined for each profile. User names and passwords are the same as the profile name You can (and should) create new users and change these passwords later, specially the admin password. You can also use other features provided by Mooshak, such as
  1. Contest results
  2. Register for on-line contest
  3. On-line help
  4. Check lists

You can also use Mooshak as a REST web service. Use this sandbox to check out how the web service is used.
