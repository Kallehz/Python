Export $type Folder: $dir

File/archive in $format format
Include DTD in XML description
Create archive for a single XML file

Note: This command exports a file/archive containing:
  • an XML file describing this folder structure and content
  • all files included in this folder

Hints:

  • You can edit the content of this content and import it back to this folder
  • Use the DTD to pre-validate the XML description. Even if the folder is empty the DTD will reflect it complete structure
